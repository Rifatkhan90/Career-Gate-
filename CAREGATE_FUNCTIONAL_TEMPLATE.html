Project Overview

CareerGate is an elite, AI-driven recruitment ecosystem designed to bridge the gap between global talent and employers. Built on a rock-solid MySQL foundation with multi-layered authentication (Native + Google OAuth) and real-time Gmail SMTP verification, the platform provides unmatched security and operational intelligence for modern talent acquisition.

Tech Stack

  • Language: JavaScript / TypeScript
  • Frontend: React.js (Vite) / Motion Designer
  • Backend: Node.js (Express) / Passport.js
  • Database: MySQL 8.0 (Relational Persistent Storage)
  • Email Gateway: Gmail SMTP for Secure OTP Delivery
  • AI Engine: Gemini AI / OpenAI API Interoperability

Functional Requirements

Module 1: Unified Identity & Secure Credential Governance

  • [Tanvir Zim] [Hybrid Auth & Encryption] Dual-stream authentication supporting standard registration and Google OAuth 2.0. All native user passwords are securely encrypted and stored using Bcrypt hashing in the MySQL database, ensuring zero plaintext exposure. Architecture: Passport.js middleware + Bcrypt.js encryption + MySQL `users` table.
    Usage: Click "Continue with Google" or standard sign-up. Admin can view the Bcrypt Hash in the dashboard.
  • [Mosaddek Alom] [SMTP-Verified Recovery] A high-security 3-step Forgot Password workflow. Generates a 6-digit OTP stored securely in MySQL and delivers it via encrypted Gmail SMTP for identity validation. Architecture: Nodemailer + MySQL otp_codes table verification.
    Usage: Login Screen -> Forgot Password -> Enter Email -> Verify OTP -> Reset Password.
  • [Rahman Tonmoy] [Master Control] Advanced Administrative dashboard with the power to Deactivate or Permanently Wipe user records from the MySQL database, instantly synchronizing changes across the platform. Architecture: Admin-privileged REST endpoints connected to MySQL DELETE/UPDATE queries.
    Usage: Admin Dashboard -> Users Tab.

Module 2: Orchestrated AI Recruitment & Talent Lifecycle

  • [Tanvir Zim] [Employer Suite] Comprehensive job management interface for employers to post, edit, and track listings stored securely in the MySQL jobs table. Includes applicant pipeline management (Interview, Accepted, Rejected). Architecture: Relational database CRUD mappings with state management.
    Usage: Employer Dashboard -> Post a New Job / View Applicants.
  • [Mosaddek Alom] [AI Talent Matching] Real-time searchable job feed with AI Readiness Scoring that calculates a percentage match between candidate profiles and job descriptions. Architecture: Algorithmic matching integrated with job search filters.
    Usage: Candidate Dashboard -> View Job Readiness Percentages.
  • [Rahman Tonmoy] [Profile Engineering] Candidate profiles including Dynamic Resume Uploads (PDF/Base64) and picture storage, fully synchronized with the MySQL profiles table. Architecture: Base64/Multipart stream to MySQL LONGBLOB data fields.
    Usage: Candidate Dashboard -> Edit Profile & Upload Resume.
  • [Tanvir Zim] [AI Resume Builder] An integrated tool allowing candidates to dynamically construct professional, ATS-friendly resumes layer by layer using AI assistance. Architecture: Contextual form state synced to profile database.
    Usage: Main Menu -> AI Resume Builder.
  • [Mosaddek Alom] [Skill Assessments] A verification module offering industry-standard testing to issue badges and scores directly to candidate profiles, increasing their trust factor. Architecture: Dynamic quiz engine mapped to candidate skill certifications.
    Usage: Main Menu -> Skill Assessments.

Module 3: Strategic Oversight, Monetization & UX Dynamics

  • [Tanvir Zim] [Revenue Hub] Multi-tier subscription model supporting Pro and Enterprise plans with integrated pricing matrix stored in BDT. Architecture: Secure checkout simulation with MySQL payment history tracking.
    Usage: Pricing -> Select Plan -> Upgrade.
  • [Mosaddek Alom] [Event Broadcaster] Automated notification engine that pushes real-time alerts for application status changes and platform-wide broadcasts from the Admin. Architecture: Event-driven reactive UI hooks.
    Usage: Notification Bell -> View Application Alerts.
  • [Rahman Tonmoy] [Professional Newsfeed] A networking wall where candidates, employers, and admins can post updates, celebrate new job placements, and interact with the community. Architecture: Interactive timeline component with real-time updates.
    Usage: Main Menu -> Newsfeed.
  • [Tanvir Zim] [AI Floating Assistant] Persistent, floating AI assistant that can summarize jobs, generate cover letters, and provide career advice based on user intent. Architecture: OpenAI-compatible LLM bridge (Gemini/Local LLM).
    Usage: Click the floating chat icon on the bottom right.
  • [Rahman Tonmoy] [Atmospheric UX] High-fidelity environmental effects (Birds, Shooting Stars, Airplanes) that react to Dark/Light mode transitions for an immersive experience. Architecture: GPU-accelerated CSS Variable animations.
    Usage: Passively active across the entire platform.
