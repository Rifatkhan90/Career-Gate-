02. Technical Implementation

Logic Modules

An audit of the specific functions and libraries that power every interactive feature of CareerGate.

@lib/store.ts / window.dispatchEvent

Reactivity Engine

Implements a custom cg_store_update event listener protocol. This enables sibling components to update their UI in real-time when a job is posted or applied to, without requiring a global context refresh.

  • Event: CustomEvent
  • Latency: < 5ms
  • Sync: Synchronous UI Update
@lib/gemini.ts / generative-ai

Neural Synthesis

Communicates directly with gemini-1.5-flash via Google Generative AI SDK. Specific prompts ensure candidate resumes are scored against job descriptions with human-like rationale.

  • Model: Pro 1.5 / Flash
  • Return: Structured JSON
  • Task: Resume Scoring / Optimization
@components/ResumeBuilder.tsx

Document Engine

Utilizes jspdf and html2canvas for document synthesis. Performs dynamic CSS-to-Canvas rasterization for professional resume exports with zero styling loss.

  • PPI: 144 DPI (Digital Master)
  • Format: ISO 216 (A4 Print)
  • Lib: Canvas2PDF Architecture
@lib/store.ts / setUserPlan

Monetization Path

Handles local state transformation for premium user tiers. Encrypts simulated plan ID into LocalStorage to persist account status across sessions.

  • Gateway: Local Payment Sim
  • Tiers: Basic / Pro / Enterprise
  • Status: persistent-active
